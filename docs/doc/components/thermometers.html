Executable¶

/bin/openheating-thermometers.py

Description¶

  • Based on a configuration, takes care about thermometers.

  • Exposes every thermometer as a D-Bus object.

  • Polls from the org.openheating.Main component initiate a temperature read.

  • Temperature reads are done in a background thread - a Onewire temperature read takes over a second.

  • Implements simulation mode where temperatures are read from files rather than real hardware. Really cool when you are on a train and the backpack is too small for an entire heating plant.
